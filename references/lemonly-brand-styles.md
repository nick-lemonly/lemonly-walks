# Lemonly Brand Styles

This document summarizes key elements of the Lemonly brand for styling purposes.

## Colors

### Primary Colors

The primary colors should be dominant in any composition. Tints and shades can be used when needed to add depth, dimension, or detail — especially shades of gray.

Lemonly Teal: #33c8c5
Lemonly Yellow: #f6d72c
Black: #1a1a1a
White: #ffffff

### Secondary Colors

Lemonly Orange: #f5b02b
Lemonly Coral: #f06445

### Neutral Colors

Black, white, and gray, are used frequently as neutrals. When using grays, we generally prefer darker and lighter shades of gray — not medium shades.

Black: #1a1a1a
White: #ffffff
Light Gray: #f9f9f9 - Often used as a page background, like on our website.

Other shades as needed, such as:

Dark Gray 1: #333333
Dark Gray 2: #666666
Medium Gray: #b3b3b3

### Color Accessibility

Accessibility is important to Lemonly, so we always strive to meet Level AA WCAG compliance. Color contrast is one important aspect of accessibility. Notably, since most of our brand colors are lighter hues, we avoid using white text on most colored backgrounds.

Body text: 4.5:1 contrast ratio
Large text (larger than 18pt): 3:1 contrast ratio
Interface elements (icons, buttons, etc.): 3:1 contrast ratio

When needed, it can be helpful to have an "accessible" variant of certain brand colors for adequate text contrast. For this purpose, we have an "accessible" shade of Lemonly Teal: #41a2a2. This variant shade should only be used when paired with white, either as a text color or background for adequate contrast.

## Typography

### Fonts

Montserrat is our primary brand font. We use various styles and weights within the Montserrat font family.

When needed, we also incorporate other brand fonts including Oswald or Steelfish when narrower fonts are needed, as well as Oooh Baby or Caveat as script fonts used as accents.

### Heading Styles

The heading styles below should be used as a baseline for styling text, especially in documents. When designing more visual pieces like graphics or interfaces, these heading styles can serve as a more general guideline but not strict requirements. On web, we prefer heavier weights for headings (Extra Bold instead of Bold).

Title: Montserrat, Bold, 24

Subtitle: Montserrat, Normal, 18

Heading 1: Montserrat, Bold, 22

Heading 2: Montserrat, Bold, 18

Heading 3: Montserrat, Bold, 14

Heading 4: Montserrat, Bold, 12

Heading 5: Montserrat, Normal, 12

Heading 6: Montserrat, Italic, 10

Body text: Montserrat, Normal, 10

### Type Spacing

Line spacing: 1.2 (or larger when needed)
Paragraph spacing; 2

### Capitalization

Lemonly almost always uses sentence case, never title case (except for filenames, document titles, or similar).

We also use UPPERCASE in some headings, such as slide titles in presentations, H1s on web pages, stylized H2s (as an alternate option on web), or other stylized text. When uppercase is used in web or coded projects, it should be applied using `text-transform: uppercase` or equivalent stylings so the text remains readable for accessibility purposes.
